SECIHTI Budget Planning

Graphical Budget Simulation Tool

This module provides a graphical interface for planning and simulating how to use remaining budgets in SECIHTI projects.

Features

  • Visual budget planning interface
  • Create planned expenses (simulated)
  • Allocate budget from multiple rubros to expenses
  • Partial or full allocation support
  • Simulation mode - doesn't affect real budgets
  • Track budget movements and allocations

How It Works

Left side: Activities and their budget lines (rubros) showing remaining budgets.
Right side: Planned expenses that need funding.
Allocations: Connect budget lines to expenses by creating allocations - each expense can be funded from multiple rubros.
